# Schema for the index pattern "*flights,-.*"

## Indices

* kibana_sample_data_flights

## Fields

Here is the combined list of fields in these indices, their type and most common values:

* AvgTicketPrice: float, mostly 4e+2 to 9e+2
* Cancelled: boolean
* Carrier: keyword, only Logstash Airways, JetBeats, Kibana Airlines, ES-Air
* dayOfWeek: integer, mostly 1 to 5
* Dest: keyword, e.g. Zurich Airport, Warsaw Chopin Airport, Verona Villafranca Airport
* DestAirportID: keyword, e.g. ZRH, XIY, YWG, WAW, VR10
* DestCityName: keyword, e.g. Zurich, Xi'an, Shanghai, Winnipeg, Tokyo
* DestCountry: keyword, e.g. IT, US, CN, CA, JP
* DestLocation: geo_point
* DestRegion: keyword, e.g. SE-BD, IT-34, CH-ZH, CA-MB, GB-ENG
* DestWeather: keyword, mostly Rain, Clear, Sunny, Cloudy, Heavy Fog
* DistanceKilometers: float, mostly 2e+3 to 1e+4
* DistanceMiles: float, mostly 1e+3 to 7e+3
* FlightDelay: boolean
* FlightDelayMin: integer, mostly 0 to 8e+1
* FlightDelayType: keyword, mostly No Delay, Late Aircraft Delay, NAS Delay, Carrier Delay, Security Delay
* FlightNum: keyword, e.g. 00882F6, 009NIGR, 00CGX81, 00DRRXR, 00HGV4F
* FlightTimeHour: keyword, e.g. 0, 3, 1.5, 5, 2.25
* FlightTimeMin: float, mostly 2e+2 to 8e+2
* Origin: keyword, e.g. El Dorado International Airport, Itami Airport
* OriginAirportID: keyword, e.g. UIO, EZE, BOG, CDG, ITM
* OriginCityName: keyword, e.g. Rome, London, Quito, Osaka, Seoul
* OriginCountry: keyword, e.g. IT, US, JP, CN, CA
* OriginLocation: geo_point
* OriginRegion: keyword, e.g. SE-BD, GB-ENG, IT-34, IT-25, IT-62
* OriginWeather: keyword, mostly Clear, Cloudy, Rain, Sunny, Thunder & Lightning
* timestamp: date
